# UI Workflow Video Script

## Goal
Create a short video that clearly shows step-by-step interaction with the Bluetooth earphone UI prototype.

## Recommended Duration
60 to 120 seconds.

## Recording Method Options
- Option A: Screen recording of HTML mockups in sequence.
- Option B: Slideshow video made from selected wireframe/mockup images with captions.
- Option C: Hybrid (screen recording + still images for low-fi references).

## Required On-Screen Label Rule
Each step should include a visible label such as: `Step 1: Open app`, `Step 2: Add device`, etc.

## Suggested Shot List (Example)

| Time | Step label | Screen / action | Source |
|---|---|---|---|
| 00:00-00:06 | Step 1: Launch | Open welcome screen and tap Get Started | `docs/mockups/welcome.html` |
| 00:06-00:18 | Step 2: Add Device | Show empty home and tap Add Device | `docs/mockups/home/empty.html` |
| 00:18-00:28 | Step 3: Pairing | Show Bluetooth searching then success transition | `docs/mockups/home/bluetooth-pairing.html` -> `docs/mockups/home/populated.html` |
| 00:28-00:40 | Step 4: Sound Setup | Open Sound tab, select Custom, adjust Bass/Mid/Treble, Save | `docs/mockups/sound/equalizer.html` |
| 00:40-00:50 | Step 5: Noise Mode | Switch to Noise tab and select Transparency mode | `docs/mockups/noise/modes.html` |
| 00:50-01:02 | Step 6: Settings | Open General -> Device Detail -> Automatic Power Off | `docs/mockups/settings/general.html`, `docs/mockups/settings/device-detail.html`, `docs/mockups/settings/power-off.html` |
| 01:02-01:14 | Step 7: Profile/Auth | Show login/register/forgot-password path and profile overview | `docs/mockups/profile/login.html`, `docs/mockups/profile/register.html`, `docs/mockups/profile/forgot-password.html`, `docs/mockups/profile/overview.html` |
| 01:14-01:20 | Step 8: End State | Return to populated home and summarize task completion | `docs/mockups/home/populated.html` |

## Narration Template (Optional)
- "In Step 1, the user starts the app and enters the home flow."
- "In Step 2 and 3, the user pairs a new Bluetooth earphone."
- "In Step 4 and 5, the user personalizes sound and listening mode."
- "In Step 6 and 7, the user manages device settings and profile actions."
- "Finally, the user returns to Home with all key tasks completed."

## Export Checklist
- MP4 format.
- 1080p recommended.
- Captions/labels visible in all steps.
- Text size readable on laptop projector.
- No need for polished visual design; interaction sequence clarity is the priority.
